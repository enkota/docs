## Your first AI sales agent

1. Navigate to the [agents](https://supmaya.com/agents) page on the dashboard.
2. Click `Create agent` and add a name.
3. Fill out your prompt script (You can use the AI prompt generator to help you).
4. Add your external booking link to show at the qualified step.
5. Add your Webhook API endpoint if you would like to send newly qualified opportunities to an external system. We also support Zapier triggers.
